# AI/ML Assignment - AccuKnox

## Overview
This project demonstrates API data retrieval, database storage, and visualization using Python.

## Features
- Fetch data from API
- Store in SQLite
- Calculate average scores
- Visualize using bar chart
- Import CSV to database

## How to Run
pip install -r requirements.txt

python api_to_sqlite.py
python data_visualization.py
python csv_to_db.py

## Assumptions
- Public APIs used
- Sample data used where required
